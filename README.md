# シェルピンスキーの三角形

[O'Reilly Japan - RustとWebAssemblyによるゲーム開発](https://www.oreilly.co.jp/books/9784814400393/)の1章のシェルピンスキーの三角形を実装したものです。

## How to install

```sh
pnpm install
```

## How to run in debug mode

```sh
pnpm start
```

## How to build in release mode

```sh
pnpm build
```
